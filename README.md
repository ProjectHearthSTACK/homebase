# HomeBase

> *You don't have to figure it out alone.*

HomeBase is a financial literacy and life skills app for young adults starting from zero. Built under Project Hearth.

## Stack
- React 18 + TypeScript
- React Router v6
- Vite
- Vercel (deploy)

## Run locally
```bash
npm install
npm run dev
```

## Screens
- `/` — Welcome
- `/situation` — Situation check-in
- `/dashboard` — Module library
- `/lesson/:moduleId` — Lesson cards
- `/profile` — Profile
